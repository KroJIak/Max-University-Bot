# Mini App Starter

Базовый шаблон на React + TypeScript + Vite для быстрого старта новых проектов.

## Скрипты

- `npm run dev` — локальная разработка с HMR
- `npm run build` — production-сборка
- `npm run preview` — предпросмотр итоговой сборки
- `npm run lint` — проверка кода ESLint

## Структура

- `src/App.tsx` — корневой компонент приложения
- `src/pages/MainPage` — стартовая страница с приветственным блоком
- `src/index.css` — минимальные глобальные стили

## Как начать

```bash
npm install
npm run dev
```

Замените контент `MainPage` и добавляйте новые модули по мере развития проекта.
